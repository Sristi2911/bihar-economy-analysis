# Bihar Economy Analysis
### Sectoral and Comparative Analysis with the Indian Economy

A Python data analysis project based on official GSDP data collected during an internship at the **Directorate of Economics and Statistics (DES), Government of Bihar, Patna**.

---

## Key Finding

Bihar has consistently outpaced India's GDP growth rate in several recent years, yet its per capita income has remained at roughly **30–35% of the national average** throughout 2011-2025. High GSDP growth is not translating into per capita convergence, pointing to structural issues in labour productivity, industrialisation depth, and formal employment generation.

## Data Sources

All data is from official government sources:
- **DES Bihar**: Gross State Value Added (GSVA) and GSDP series
- **MoSPI**: National Accounts Statistics, India GDP series
- Base year: **2011-12** (MoSPI methodology)
- Coverage: **2011-12 to 2024-25** (2023-24 provisional, 2024-25 quick estimates)

## How to Run

1. Download all the project files and keep them in the same folder structure as shown above.
2. Install the required Python libraries by running this once in your terminal:
   ```
   pip install pandas matplotlib numpy jupyter
   ```
3. Open the notebook: launch Jupyter, navigate to the project folder, and open `bihar_economy_analysis.ipynb`.
4. Run the cells from top to bottom. Charts will be saved automatically to the `charts/` folder.

## Background

This analysis was conducted as part of an internship at the **Directorate of Economics and Statistics (DES), Patna,** the central statistical authority of Bihar, responsible for compiling the state's official GSDP estimates under MoSPI guidelines.

The internship involved working directly with Bihar's GSVA datasets, understanding state income estimation methodology, and conducting sectoral comparisons with India's national accounts.
